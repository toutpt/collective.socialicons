Introduction
============

This addon provide differents sets of social media icons.
It provides also python data structure to get path of each icons.
::

    {'3d-icons-designbolts':
      {'facebook':
        {'64':'64/facebook.png', '32': '32/facebook'},
        ...
       'twitter':
       ..
      },
